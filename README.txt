1. Для разворачивания проекта на компьютере необходимо иметь [NodeJS с его пакетным менеджером - npm](https://nodejs.org/en/download);
2. Так же необходим [AngularCLI](https://angular.io/cli);
3. В корне проекта необходимо загрузить все зависимости командой: /npm install ;
4. В корне проекта надо запустить angular-сервер следующей командой: /ng serve --port 4200 --host localhost --open ;
- Эта команда запустит билд приложения на локальном сервере по адресу http://localhost:4200 .
- Так же она автоматически откроет этот адрес в стандартном браузере после успешного запуска.
